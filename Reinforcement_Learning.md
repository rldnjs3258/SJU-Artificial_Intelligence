# 1. Reinforcement Learning
 - 에이전트가 올바른 행동을 하면 보상(reward)를 주고, 그릇된 행동을 하면 처벌(penalty)를 줌으로써 에이전트가 올바르게 행동하게 한다.
 - 강화학습은 올바른 행동결과에 대해서도 고려한다.
